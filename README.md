## Como executar a aplicação?
```sh
  # Clone o projeto
  git clone git@github.com:EmmanuellaAlbuquerque/warehouse-vue.git

  # Instale as dependências
  yarn install

  # Inicie o servidor de desenvolvimento
  yarn serve
```
